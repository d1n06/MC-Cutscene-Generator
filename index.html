Minecraft Cutscene Generator

In this page you can create your own cutscenes for minecraft in the form of a data pack.

There are a lot of features to be added to this tool yet. Right now, only one cutscene can exist in each minecraft world.
You'll have to be on spectator mode or flying in creative mode for the cutscene to work correctly.

To install the cutscene, press the download button and save the .zip file in the "datapacks" of your world's folder (or follow this guide).
